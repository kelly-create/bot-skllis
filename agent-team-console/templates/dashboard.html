示例：python3 scripts/run_tests.py ｜ npm run build ｜ pytest -q。 建议为每个任务提供可重复执行的命令，便于回归和重试。
操作提示（中文）
  • 启动：把任务放入执行队列；并发满时会自动排队。
  • 停止：中止当前任务并标记为 failed（返回码 143/137）。
  • 重置：把 done/failed 任务恢复成 pending，便于二次执行。
  • 任务详情页：可看完整日志，页面每 4 秒自动刷新。
  • 状态说明：pending=待执行，running=执行中，done=成功，failed=失败。
